# 仿官釉青瓷筆筒

## 基本資訊
- **英文品名**: Brush holder in light green glaze imitating Guan ware
- **統一編號**: 故瓷017784N000000000
- **分類**: 陶瓷器
- **時代**: 清（1644-1911年）
- **尺寸**: 高9.8公分 口徑7.9公分 底徑7.7公分
## 文物說明
筆筒，圓口、直壁、足內凹。釉色粉青，有數道淺黃細長開片。釉厚，口緣微見胎色。滿釉，僅底緣一周露黑胎。底刻乾隆四十八年(1783)楷銘︰「缾椀官窯亦恒見，筆筒一握見殊常。宣和書畫曾經伴，南渡兵戈幸未亡。火氣全消文氣蔚，今人如挹古人芳。不安銅膽插花卉，拈筆吟當字字香。乾隆癸卯新正御題。」印︰「古香」、「太璞」。
## 導覽說明
探春的光明磊落，亦體現在興趣與品味上。「案上磊著各種名人法帖，並數十方寶硯，各色筆筒，筆海內插的筆如樹林一般。」她的字寫得極好，喜愛雄渾、端正的顏真卿。她用的筆筒，或許可見仿官釉、釉色勻淨純正的筆筒。
