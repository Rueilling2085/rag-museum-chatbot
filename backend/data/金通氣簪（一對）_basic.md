# 金通氣簪（一對）

## 基本資訊
- **英文品名**: Pair of gold hairpins with auspicious symbols
- **統一編號**: 故雜004768N000000000
- **分類**: 雜項
- **時代**: 清（1644-1911年）
- **尺寸**: 長11.8公分 簪首寬1.9公分；長11.7公分 簪首寬3.1公分
## 文物說明
金簪一對，簪首為梅花，花芯嵌紅寶石，瓣嵌珍珠，周飾纍絲葉片；鋌為纍絲鏤空管狀。
## 導覽說明
簪，插在髮髻上的飾品。固定髮髻之餘，也起到裝飾及象徵身份之用。通氣簪，簪腳金線重疊交錯，呈鏤空管狀，予人通透、透氣之感。簪首為梅形，花芯嵌紅寶，花瓣嵌珍珠，周飾纍絲葉片，刻畫出富貴之貌，應為后妃所有。
