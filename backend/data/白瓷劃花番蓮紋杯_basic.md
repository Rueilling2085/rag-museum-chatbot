# 白瓷劃花番蓮紋杯

- **統一編號**: 故瓷008334N000000000

## 導覽說明
寶釵茶杯的釉色，應該是不引人注目的白色；形制，應該是簡練低調的樣子。在妙玉給寶玉品茶的「綠玉斗」面前，想必顯得黯淡，不起眼。此杯外壁，淺淺地刻劃若有似無的蓮花；不經意可能就會忽略的蓮花，一點也不張揚。
