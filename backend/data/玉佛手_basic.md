# 玉佛手

## 基本資訊
- **英文品名**: Jade Buddha’s Hand Citron
- **統一編號**: 故玉002886N000000000
- **分類**: 玉器
- **時代**: 清 (1644-1911年)
- **尺寸**: 寬12.9公分 通高18.1公分
## 文物說明
玉質，整器雕作成一佛手瓜，瓜表縱痕甚深，至瓜頂處分裂為如指爪的瓜體，嫩梢卷曲而飽滿，萌發生意。除了做為主體的佛手瓜外，在佛手瓜下半部還雕附了小佛手瓜、桃樹、靈芝，添加了器物的吉祥寓意。
## 導覽說明
佛手，佛伸出慈悲之手。賈家的慈悲與善念，拯救貧困的姥姥一家。姥姥一家的感恩與懷念，拯救落難的巧姐。佛手「千里伏線」，串連整部小說。而寶玉，看破紅塵、俗緣已畢後，最終出家，回歸原處，回歸成「通靈寶玉」。
