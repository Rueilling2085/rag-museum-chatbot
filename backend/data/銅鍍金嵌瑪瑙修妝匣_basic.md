# 銅鍍金嵌瑪瑙修妝匣

## 基本資訊
- **英文品名**: Gilt bronze necessaire decorated with agate inlay
- **統一編號**: 故雜000653N000000000
- **分類**: 雜項
- **時代**: 西元18世紀
- **尺寸**: 長6.1公分 寬3.7公分 高10.9公分
## 文物說明
長方拱形蓋匣，鑲瑪瑙與鍍金花環奔鹿邊飾，底部四角配置獅足托座。內匣盛裝橢圓小鏡，切稜旋鈕透明玻璃方瓶，刀、剪、匙，及鉛筆、書寫用象牙片。此類袖珍組件匣盒，裝盛物件多包括修容理妝用品，流行於十八世紀歐洲上層社會。
## 導覽說明
無論金星玻璃或一般玻璃，在清宮皆屬於珍稀的舶來品。除了金星玻璃，《紅樓夢》還提到「玻璃小瓶，卻有三寸大小，上面螺絲銀蓋！」袖珍匣盒盛放之玻璃方瓶，正因小巧不及三寸，更顯得切稜旋鈕、瓶塞及金屬蓋之精緻。
