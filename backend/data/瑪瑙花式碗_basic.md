# 瑪瑙花式碗 

## 基本資訊
- **英文品名**: Floral-shaped agate bowl
- **統一編號**: 故雜000108N000000000
- **分類**: 雜項
- **時代**: 清 雍正 (1723-1735年)
- **尺寸**: 通高3.2公分 口徑11.4公分 底徑3.9公分
## 文物說明
瑪瑙質，黃褐色，有紅斑與黑色渦紋。器作六瓣花形，器壁薄透，侈口斂腹淺圈足，全器光素無紋，圈足琢有「雍正年製」四字篆款。
## 導覽說明
由於瑪瑙質地堅硬，有著天然而華麗的色澤與紋路，自古以來便經常被用來製作各種精美的生活器具如杯、碗器，身上的飾品，或是典禮所使用的禮器。在明清兩代，瑪瑙器的製作更加盛行，並發展出一套鑑賞的標準和品味。例如清代雍正時期製作的瑪瑙器，大多造型簡潔工整、色澤樸實素雅、線條勻稱精細，與皇帝個人的審美品味有很大的關聯。\\n這件以瑪瑙製作而成的小碗，碗的外側底部刻有篆書字體「雍正年製」款識。由於雍正皇帝只會在他覺得好的作品上留下年款，所以這件作品應該是符合皇帝心中素淨文雅的佳作。碗身做成六個花瓣的造型，線條勻稱流暢，碗壁輕巧薄透，整體呈現黃褐色，並帶有天然的紅色斑紋和黑色的圓形水渦紋。碗身沒有過多的裝飾，而是將瑪瑙石材自然樸拙的美感發揮地淋漓盡致。在清代小說《紅樓夢》中，就曾數次提到賈府所用的瑪瑙器物，而有一回，賈寶玉命人將荔枝盛裝在一只纏絲白瑪瑙碟子中，充分展現出清代勳貴人家講究的器物美學。
