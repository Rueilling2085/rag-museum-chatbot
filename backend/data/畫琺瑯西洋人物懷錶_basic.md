# 畫琺瑯西洋人物懷錶

## 基本資訊
- **英文品名**: Pocket watch with Western figures decoration in painted enamels
- **統一編號**: 故雜007132N000000000
- **分類**: 雜項
- **時代**: 清 (1644-1911年)
- **尺寸**: 錶身徑3.7公分 厚1.1公分
## 文物說明
圓形指針式懷錶，為合金質，器身厚實。錶盤作金底，上以黑色等距書寫羅馬數字I(一)至XII(十二)，在羅馬數字外側加圓點一週標示六十個時間刻度，內側飾一圈花葉紋；長短指針固定於錶盤正中心，最細長者為秒針，次長者為分針，短者為時針。背面以畫琺瑯技法彩繪西洋林間風景，遠方見城堡塔樓，畫面正中為一位騎馬的西洋女性，右手執韁繩、左手伸出停一獵鷹，另有一小童牽著獵犬步行隨侍在側，二人目光集中在負責探查之獵鷹身上。錶上方栓一U形吊環，原用以接錶鏈處，改繫黃色彩穗，穗帶同時穿過鐘錶所附之發條鎖匙。錶正反面外圍及吊環均嵌飾珍珠。此器機件及鑰匙有銹蝕，彩穗髒污，部分嵌件脫落。懷錶源自西洋，通常以錶鏈繫於衣物、便於隨身攜帶，不使用時可收藏在口袋中，故名為&quot;pocketwatch&quot;(口袋錶)。
## 導覽說明
如果曹雪芹見到此錶，可能會如此描述：「表面是個西洋琺瑯的棕髮赤身男孩，兩肋又有肉翅，騎在魚的背上。」事實上，是海神帕萊蒙騎著海豚，在海上守護著船舶。或許正因不理解，反倒增添人們對此類物品的獵奇與興趣。
