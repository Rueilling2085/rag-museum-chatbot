# 玉鎖形佩

## 基本資訊
- **英文品名**: Jade pei pendant in the shape of a lock
- **統一編號**: 故玉006682N000000000
- **分類**: 玉器
- **時代**: 清 (1644-1911年)
- **尺寸**: 長8.2公分 寬5.3公分
## 文物說明
玉質，全器琢成一如意雲形鎖，一面浮雕梅花數朵與飛禽，鳥之冠、羽毛雕刻細膩。背面浮雕四字「無壽萬年」。
## 導覽說明
「不離不棄，芳齡永繼。」兩句吉讖是一位和尚給寶釵的，說要鏨在金器上，正、反面各一句。此類呈鎖片形，刻吉祥語之作，並非憑空想像。如此玉佩，外觀即鎖形，一面牡丹，一面「玉堂富貴」。以絡子絡上，得以隨身佩戴。
