# 金碗

## 基本資訊
- **英文品名**: Gold bowl
- **統一編號**: 故雜001843N000000000
- **分類**: 雜項
- **時代**: 清 同治（1862-1874年）
- **尺寸**: 口徑16.0公分
## 文物說明
金碗，撇口，深壁略下斂，圈足。整器光素無紋，通體金黃，頗見富貴。
## 導覽說明
《皇朝禮器圖式》所確立的禮制與規範，成為後代各朝行事的參考與依據。金碗，亦稱金盌，顯得金光燦爛。鵝黃籤上寫著：「金碗（盌）一件，係八成金重十二兩五分」。含金量之高，象徵著帝后的身份，襯托著帝后的富貴。
