# 銀嵌金星玻璃鼻煙盒

## 基本資訊
- **英文品名**: Silver snuff box with aventurine and glass inlay
- **統一編號**: 故雜000869N000000000
- **分類**: 雜項
- **時代**: 西元18世紀
- **尺寸**: 縱長7.2公分 寬5.4公分 高1.5公分
## 文物說明
磨邊金星玻璃掀蓋盒，蓋盒嵌銀口，金星玻璃品質極佳，閃爍著低調迷人的光芒，盒蓋密合做工精緻。
## 導覽說明
鑲嵌金星玻璃的鼻煙盒，除有銅鍍金之外，還有銀製、銅纍絲等作品。此類鼻煙盒，盒蓋與盒身大多設有樞軸，得以上掀開闔。此銀製八方盒，盒蓋鑲嵌之金星玻璃，磨邊帶有稜面，製作精巧、細緻，並閃爍著如星點般的顆粒。
