# 瑪瑙扭絲紋碟

## 基本資訊
- **英文品名**: Agate dish with skein pattern
- **統一編號**: 故雜000332N000000000
- **分類**: 雜項
- **時代**: 清 乾隆（1736-1795年）
- **尺寸**: 高2.7公分 口徑10.5公分 足徑3.15公分
## 文物說明
半透明橙、赭、白交雜瑪瑙質，色彩斑斕鮮豔。侈口，斜壁，淺圈足，全器雕琢極薄，器外壁淺浮雕紐絲紋。底琢「乾隆年製」四字篆款。
## 導覽說明
扭絲紋一道接著一道，迴旋向著碟心積聚。人為雕琢的扭絲，與自然擁有的紋理，一同在碟中流動、暈染開來，使碟子彷彿有了生命。若能挑揀若干取得不易的鮮荔枝置於其上，如此更能映襯、凸顯瑪瑙紅如錦繡的結晶紋理。
