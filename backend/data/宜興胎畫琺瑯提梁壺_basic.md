# 宜興胎畫琺瑯提梁壺

## 基本資訊
- **英文品名**: Yixing teapot with a loop handle in painted enamels
- **統一編號**: 故瓷017560N000000000
- **分類**: 陶瓷器
- **時代**: 清 康熙（1662-1722年）
- **尺寸**: 寬7.0公分 通高7.8公分 蓋高2.7公分
## 文物說明
平口，圓碩壺身，平底下接三足，短流，提梁極長，蓋鈕塑黃色奇獸一隻。器用宜興紫砂泥地，胎質不似瓷泥細膩，顆粒較粗，胎中間有黑、黃細砂點，造成表面凹凸不平，故紋飾釉彩處常見細小氣泡劃破所留棕眼遺痕。腹精繪菊花、牡丹與品字形裝飾圖案。壺底書「康熙御製」二行四字黃料楷款，外加框圈一周。有木匣附件一。
## 導覽說明
楠木製成的木匣，刻寫著「康熙年製宜興畫琺瑯提梁壺一件」，說著文物的名字，也說著乾隆皇帝著手整理由康、雍、乾三朝的畫琺瑯製品，逐一配匣並集中收藏的故事。小提梁壺不盈一握，可能是文房玩器，也可能是案頭硯滴。
